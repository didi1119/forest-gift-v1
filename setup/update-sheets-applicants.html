新增大使申請管理表 (Applicants)

🆕 新增功能：完整的大使申請審核流程

步驟1: 新增 Applicants 工作表

在您現有的 Google Sheets 中新增一個名為 "Applicants" 的工作表，並設定以下標題行：

📋 Applicants 工作表（大使申請管理）

在 A1 開始輸入以下標題（一行一個欄位）：

id name email line_name phone message application_status review_notes reviewed_by partner_code_assigned partner_link_sent created_at reviewed_at approved_at
id name email line_name phone message application_status review_notes reviewed_by partner_code_assigned partner_link_sent created_at reviewed_at approved_at

步驟2: 欄位詳細說明

基本資料欄位

  • id: 自動編號
  • name: 申請者姓名
  • email: 申請者Email
  • line_name: LINE顯示名稱
  • phone: 聯絡電話
  • message: 申請留言

審核管理欄位

  • application_status: PENDING / APPROVED / REJECTED
  • review_notes: 審核備註
  • reviewed_by: 審核者
  • partner_code_assigned: 分配的大使代碼
  • partner_link_sent: 是否已發送連結 (TRUE/FALSE)
  • created_at: 申請時間
  • reviewed_at: 審核時間
  • approved_at: 核准時間

步驟3: 申請狀態管理

📋 申請流程狀態

PENDING

待審核

新申請，等待人工審核

APPROVED

已核准

通過審核，可發送連結

REJECTED

已拒絕

不符合要求，拒絕申請

步驟4: 範例資料（可選）

步驟5: 申請審核工作流程

1

用戶提交申請

透過 invitation.html 提交申請表單

2

系統記錄申請

自動新增到 Applicants 表，狀態設為 PENDING

3

人工審核

透過後台管理系統查看並審核申請

4

核准並生成連結

核准後透過連結生成器建立大使專屬連結

5

轉為正式夥伴

資料同步到 Partners 表，開始追蹤績效
